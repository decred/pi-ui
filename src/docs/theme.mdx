# Theming

## Create a custom theme

The recommended way to create a custom theme is to create a file named `theme.js`, you can add your custom values and override existing ones.

```js
// theme.js

export default {
  "color-primary": "red",
  "btn-color": "var(--color-primary)"
};
```

## Use your theme in your application

Pi-ui exposes an `useTheme` hook that accepts a `theme` argument, updates all CSS properties accordingly and
returns array with two items `[theme, setTheme] // current theme and a funtion to set new theme`


```js
import React from "react";
import { render } from "react-dom";
import { useTheme } from "pi-ui";
import customTheme from "./theme";

const App = () => {
  const [theme, setTheme] = useTheme(customTheme);
  return (
    // App jsx
  )
};

render(<App />, document.getElementById("root"));
```

## Default theme

There are two default themes, `defaultLightTheme` and `defaultDarkTheme`. They are both name exported from pi-ui.

```js
import { defaultLightTheme, defaultDarkTheme } from "pi-ui";
```

## Use theme property

You can use any property from a theme in your code:

```js
import React from "react";
import { useTheme, defaultLightTheme } from "pi-ui";

const Test = () => {
  const [theme, setTheme] = useTheme(defaultLightTheme);
  /*
  theme = {
    "color-primary": "#2970ff",
    "color-primary-dark": "#091440",
    "color-primary-light": "#2ed8a3"
    ...
  }
  */
}
```

## Use setTheme property

You can update theme by calling `setTheme` function exposed by `useTheme`hook:

```js
import React from "react";
import { useTheme, defaultLightTheme, defaultDarkTheme } from "pi-ui";

const themes = {
  light: defaultLightTheme,
  dark: defaultDarkTheme
};

const App = () => {
  const [, setTheme] = useTheme(themes.dark);
  const [themeMode, setThemeMode] = useState("dark");
  const handleToggleTheme = () => {
    if (themeMode === "light") {
      setThemeMode("dark");
      setTheme(themes.dark);
    } else {
      setThemeMode("light");
      setTheme(themes.light);
    }
  }
  return (...);
};
```

* Note that you **have** to pass a theme to `useTheme` everytime if you are using a custom theme, otherwise it will be set to the `defaultLightTheme`.
