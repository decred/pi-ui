## Usage

### Simple link

  <Link href="#">Click me</Link>
</Playground>

### Simple grey link
<Playground>
  <Link gray href="#">Click me</Link>
</Playground>

### Truncated link
<Playground>
  <Link truncate href="#">I am a huuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuge link!</Link>

### Link with a custom element
This is useful for specifying a custom element such as a React-Router Link

<Playground>
  <Link href="#" customComponent={(props) => <div {...props} />}>Click me</Link>
</Playground>
